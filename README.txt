MicrobesAndPhenology

This is analysis code, input, and output files associated with the manuscript:
O’Brien, AM, N Ginnan, M Rebolleda-Gómez, MR Wagner. Microbial Effects on Plant Phenology and Fitness. American Journal of Botany. 
and its preprint on EcoEvoRxiv, doi:10.32942/osf.io/exadg

The code file is annotated. Please see manuscript for more information on the contents of this repository.
